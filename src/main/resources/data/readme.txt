Spring AI Agent - Data Directory
=================================

This directory contains data files that the AI agent can read and search.

Usage:
- Add any .txt, .md, or other text files here
- Ask the AI to "list files", "read <filename>", or "search for <keyword>"
- The AI will automatically use these files to answer your questions

Sample files included:
- products.txt  : sample product catalog
- faq.txt       : frequently asked questions

Tip: You can configure a different data directory by setting the
     DATA_DIR environment variable before starting the application.
